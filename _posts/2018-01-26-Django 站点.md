### 环境准备
1）CentOS Linux release 7.2.1511 (Core)
2) Python 2.7或者以上

### 部署安装
```
[root@biezz ~]# yum -y install epel-release
[root@biezz ~]# yum -y install python-pip
[root@biezz ~]# python -m pip install --upgrade pip
[root@biezz ~]# pip install Django==1.8.16
```
