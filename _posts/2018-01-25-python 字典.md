### 字典的定义  
```a = {1:2}```  
  
1--->key   2--->value   键值是唯一的，值可以是任意数据结构，无序的哈希，json串  

### 字典的访问（查询）
```
a.get(1)
2
a[1]
2 
```
总结：a.get(key); a[key]

### 增删改

```
a = {1:2,3:4,5:6}
```

### 字典的遍历
```
a = {1:2,3:4,5:6}
a.keys()
[1, 3, 5]
a.values()
[2, 4, 6]
a.items()
[(1, 2), (3, 4), (5, 6)]
```

